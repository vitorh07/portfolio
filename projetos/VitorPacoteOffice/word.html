Microsoft Word

O Microsoft Word 2010 é um programa de processamento de texto, projetado para ajudá-lo a criar documentos de qualidade profissional. Com as melhores ferramentas de formatação de documento, o Word ajuda a organizar e escrever os documentos com mais eficiência.

Tecla de Atalho Ação
Ctrl + O Abrir novo documento em branco
Ctrl + D Alterar formatação de caracteres
Ctrl + B Salvar como
Ctrl + N Negrito
Ctrl + T Selecionar o documento inteiro
Ctrl + V Colar
Ctrl + C Copiar
Ctrl + Z Anular a última ação
Ctrl + P Imprimir
Ctrl + S Sublinhar texto
