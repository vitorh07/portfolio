Microsoft Excel

O Microsoft Excel é um editor de planilhas produzido pela Microsoft para computadores que utilizam o sistema operacional Microsoft Windows, além de computadores Macintosh da Apple Inc. e dispositivos móveis como o Windows Phone, Android ou o iOS.

Tecla de Atalho Ação
↑ Movimenta o seletor uma célula para cima
→ Movimenta o seletor uma célula para a direita
← Movimenta o seletor uma célula para a esquerda
Ctrl + N Negrito
Ctrl + T Selecionar toda a planilha
Ctrl + V Colar
Ctrl + C Copiar
Ctrl + Z Anular a última ação
Ctrl + P Imprimir
Ctrl + S Sublinhar texto
